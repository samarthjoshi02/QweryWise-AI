# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-21
### Added
- Initial release of QweryWise-AI Enterprise Self-Correcting RAG Platform.
- Multi-agent LangGraph workflow for query processing.
- Next.js frontend with Tailwind CSS and shadcn/ui.
- FastAPI backend with async architecture.
- PostgreSQL, Qdrant, MinIO, Redis integrations via Docker Compose.
- Confidence scoring and groundedness validation for LLM responses.
